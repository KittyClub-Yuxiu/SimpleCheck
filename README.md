# SimpleCheck

一个简洁优雅的任务管理应用，帮助您高效管理日常待办事项。
*by 聿修*

## ✨ 功能特点

- 📝 **任务管理**：添加、编辑、删除和标记完成任务
- 📅 **任务类型**：支持普通任务和每日重复任务
- 💬 **一言系统**：自动加载励志名言
- ⚙️ **个性化设置**：自定义应用行为和外观
- 💾 **数据持久化**：本地存储任务数据，确保数据安全
- 🎨 **精美UI**：现代化、简洁的用户界面设计

## 🛠️ 技术栈

- **开发框架**：WPF (Windows Presentation Foundation)
- **编程语言**：C#
- **架构模式**：MVVM (Model-View-ViewModel)
- **数据持久化**：本地文件存储
- **UI设计**：XAML + 自定义样式


## 🚀 安装和运行

到项目的releases处下载压缩包，双击同名exe文件即可运行！

## 📖 使用说明

### 添加任务
1. 点击添加按钮 (+)
2. 填写任务标题、描述和时间
3. 选择任务类型（普通/每日）
4. 点击保存

### 管理任务
- **标记完成**：点击任务前的复选框
- **删除任务**：点击右侧清除按钮（鼠标移动过去才会显示）

### 自定义设置
点击右上角设置按钮，可调整应用的各项设置。
